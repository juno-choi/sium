# 🎨 UI/UX 디자인 설계

## 디자인 컨셉

**핵심 키워드**: 귀여움, 게임화, 성취감, 친근함

### 컬러 팔레트

```css
/* Primary - 메인 퍼플/바이올렛 톤 */
--primary-50: #f5f3ff;
--primary-100: #ede9fe;
--primary-400: #a78bfa;
--primary-500: #8b5cf6;
--primary-600: #7c3aed;

/* Secondary - 따뜻한 핑크/코랄 */
--secondary-400: #fb7185;
--secondary-500: #f43f5e;

/* Success - 완료/성공 */
--success-400: #4ade80;
--success-500: #22c55e;

/* Warning - 경고/주의 */
--warning-400: #facc15;
--warning-500: #eab308;

/* Background - 다크 모드 지원 */
--bg-light: #fefefe;
--bg-dark: #0f0f1a;
--bg-card-light: #ffffff;
--bg-card-dark: #1a1a2e;
```

## 애니메이션 & 인터랙션

### 경험치 획득 애니메이션

```
+20 XP ✨
   ↑
   │ (위로 떠오르며 페이드아웃)
   │
 [Todo 항목]
```

### 레벨업 연출

```
┌─────────────────────────────────┐
│                                 │
│     🎉 Level Up! 🎉             │
│                                 │
│        🐱 → 🐱✨                 │
│    푸딩이가 성장했어요!          │
│                                 │
│        Lv.4 → Lv.5             │
│                                 │
│         [ 확인 ]                │
│                                 │
└─────────────────────────────────┘
```

### 마이크로 인터랙션

| 요소 | 인터랙션 |
|------|----------|
| 버튼 호버 | scale(1.02) + 그림자 확대 |
| 카드 호버 | translateY(-4px) + 그림자 |
| 체크박스 | 체크 시 스프링 애니메이션 |
| 경험치 바 | 증가 시 부드러운 확장 |
| 캐릭터 | idle 상태에서 미세한 움직임 |

## 반응형 디자인

### 브레이크포인트

| 디바이스 | 너비 | 레이아웃 |
|----------|------|----------|
| 모바일 | < 640px | 1열, 하단 네비게이션 |
| 태블릿 | 640px - 1024px | 1-2열 |
| 데스크톱 | > 1024px | 2-3열, 사이드바 |

### 모바일 우선

- 터치 친화적인 버튼 크기 (최소 44x44px)
- 스와이프 제스처 지원 고려
- 하단 고정 액션 버튼
